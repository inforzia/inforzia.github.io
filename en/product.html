Flomon is an intelligent IoT platform that securely connects and monitors various corporate devices and external data, ultimately helping businesses gain new insights. Through the browser-based Flow Engine and easy AI diagnostic design, it can be easily applied to complex IT and IoT environments, allowing businesses to fully adapt to rapidly changing demands.
CONNECTIVITY
Quick and Simple Integration
Flomon provides various built-in nodes to
simplify the connection and interface
of industrial sites. By using these nodes,
you can easily connect devices and data,
enabling fast processing
from data collection to AI diagnostics.
WIDGET DASHBOARD
Intuitive Data
The widget-based dashboard helps
facilitate advanced data analysis with ease.
It visualizes rapidly changing data and AI
analysis results, allowing
you to view large amounts of data at a glance.
NOTIFICATION
Real-Time Alerts
The notification management feature is built-in, allowing
users to easily configure settings and receive
real-time notifications. Additionally, it enables
defining notifications from a service perspective,
allowing even complex situations to be managed.
SETTING
Other Management
Flomon includes additional management features
such as user permissions and security.
It allows for various configurations,
from system settings including license management
to access control for granting user permissions.
FLOW ENGINE
Create the Flow Yourself
The Flow Engine built into Flomon allows you to create data processing workflows in the browser without coding. Using the built-in nodes, it helps collect data and makes it easier to analyze. Additionally, the integrated Jupyter Notebook and AI nodes enable seamless integration with other AI services.
MONITORING
Real-Time Monitoring
Easily manage data groups and view performance, cost, and alarm information. You can flexibly select the monitoring period, and the data remains fixed even when switching screens, ensuring continuity in data analysis.
